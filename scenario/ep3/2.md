# Gitcourse课件图片插入
您可以直接将图片内置在gitcourse项目中  
您也可以使用对象存储上传图片，并在上传完图片后获取到超链接并以MarkDown的格式引入到课件中  
`![标题](链接)`  
```
- 项目根目录
|- course.json
|- scenario
    |-- ep3
        |-- 0.md
        |-- 1.md
    |- images
	|-- 1.png
	|-- 2.png
|- ...其它文件或者目录
```

在本示例中，访问文件目录中的*2.png*的正确做法是:  
`![](../images/2.png)`
